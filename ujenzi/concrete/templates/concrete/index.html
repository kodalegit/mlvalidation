Welcome to Ujenzi!

Ujenzi is a platform that enables the use of machine learning to predict compressive strength of concrete from a number of input parameters without the need to wait for laboratory testing. The machine learning model is trained and validated using 1030 data samples. The model attains an R2 value of 0.96 and a mean-squared error of 11.486 MPa. View technical documentation for more details.
Perform predictions on compressive strength by providing input parameters such as content of cement, water, coarse aggregate, fine aggregate, blast furnace slag, superplasticizer and fly ash.
